O ecossistema Claude:
3 modos, 1 ferramenta

A Claude existe em 3 formatos diferentes. Cada um tem um propósito. Entender isso muda tudo.

Modo 1 · claude.ai
Chat
O básico. Você pergunta, ela responde. Ótimo para testar e aprender. Funciona pelo navegador, sem instalar nada.
Modo 2 · App Desktop
Cowork
A Claude com acesso às suas ferramentas: Google Drive, Docs, pastas do computador. Analisa, cria e entrega arquivos reais.
Modo 3 · App Desktop ✦
Claude Code
O ambiente mais poderoso — cria skills, agentes e sistemas personalizados. É onde vamos trabalhar hoje.
Sobre o terminal: existe uma versão via linha de comando que é o ambiente mais avançado para desenvolvedores — mais liberdade, mais poder, mais criações possíveis. É incrível, mas é mais técnico. Para tudo que você precisa no consultório, o app desktop já resolve com louvor.
Como instalar
  • 1
    Acesse claude.ai e crie sua conta
  • 2
    Baixe o Claude Desktop App — disponível para Mac e Windows
  • 3
    Faça login com a mesma conta — todas as conversas ficam sincronizadas
  • 4
    No app, ative o Claude Code e conceda as permissões solicitadas

Bloco 3

O que são Skills

Uma skill é um processo que você ensina uma vez e a Claude executa para sempre, sem precisar explicar de novo.

Analogia do consultório: é como um protocolo de atendimento. Você escreve uma vez, treina uma vez — e executa sempre igual. A skill é o protocolo da sua I.A.

De onde vêm as skills?
Fonte 1 · Catálogo oficial
claude.com/plugins

Skills validadas pela própria Anthropic. Qualidade garantida, instalação simples pelo app desktop.

Fonte 2 · Comunidade curada · Destaque skills.sh

O principal repositório curado de skills da comunidade. Centenas de skills prontas, validadas e com dados reais de instalação. Skills relevantes para sua área:

social-content copywriting seo-optimizer canvas-design video-editor frontend-design

Tem até skills de edição de vídeo — roteiro, corte, legenda automática e até sugestão de trilha sonora para reels e shorts. Tudo dentro do app.

Você nunca mais precisa criar do zero o que já existe.

Fonte 3 · Você mesmo
Criar a sua própria skill

A skill mais poderosa é a que você cria para o seu processo, com sua linguagem, para seus pacientes. É mais simples do que parece — vamos criar uma ao vivo.

Você não precisa aceitar a skill como ela veio. Instala uma do skills.sh, usa por alguns dias, identifica o que poderia ser diferente para a sua realidade — e pede para a Claude ajustar. Em minutos você tem uma versão personalizada que ninguém mais tem.
Como criar uma skill do zero
  • 1
    Descreva o processo — qual tarefa você quer delegar? Ex: "criar legenda de post sobre procedimento"
  • 2
    Defina o contexto — quem é você, qual seu público, qual o tom que você usa nas redes
  • 3
    Dê exemplos — mostre 2–3 exemplos de como você quer o resultado final
  • 4
    Salve como skill — a Claude empacota e você usa com um comando simples sempre que precisar
Skills são os tijolos. Você pode comprar tijolos prontos (skills.sh) ou fabricar os seus. No final da aula vamos criar skills ao vivo para as suas dores reais.
Bloco 3.5

Testar, iterar e
evoluir uma skill

Uma skill não precisa nascer perfeita. O processo de melhorar é parte do trabalho — e é mais rápido do que parece.

O ciclo de melhoria
  • 1
    Use a skill em uma tarefa real — não em um exemplo fictício. Use no seu trabalho do dia a dia agora.
  • 2
    Observe o que ficou bom e o que ficou estranho — tom errado? faltou algo? saiu com formato diferente do que você queria?
  • 3
    Descreva o que precisa mudar — fale para a Claude como se fosse para um colaborador: "o resultado ficou muito formal, quero mais próximo"
  • 4
    Peça para atualizar a skill — a Claude ajusta as instruções e salva a versão melhorada automaticamente
  • 5
    Repita nas próximas semanas — após 3–4 ciclos, a skill já está tão afinada que entrega exatamente o que você quer, toda vez
O atalho mais poderoso
Truque de ouro
Faça a tarefa → peça para virar skill

Você executa uma tarefa normalmente na Claude — escreve uma legenda, monta um orçamento, cria um roteiro de vídeo. O resultado ficou bom. Em vez de anotar o que fez, você simplesmente pede:

Prompt
O resultado ficou ótimo. Crie uma skill baseada no processo que você usou agora para que eu não precise explicar tudo de novo nas próximas vezes. Dê um nome curto e descritivo para ela.

A Claude analisa o que foi feito, extrai o padrão e cria uma skill pronta para reutilizar. Você não precisa lembrar nada — ela guarda o processo por você.

Quando uma skill está pronta de verdade?
  • Você a usa sem precisar dar instruções extras
  • O resultado já sai no tom e formato que você quer
  • Você consegue delegar para qualquer pessoa usar — mesmo sem explicar nada
  • Você para de pensar "como peço isso?" e começa a pensar só no resultado

Skill instalada para conteúdo

Como instalar no Claude Code

Cria posts, legendas, roteiros de carrossel e conteúdo para qualquer rede social — só texto, sem ferramenta externa. Adapta tom, voz e objetivo conforme o contexto que você definir.
Skill 1 · Legenda de post
Crie uma skill com isso
Crie uma skill chamada "legenda-harmonizacao" que escreva legendas para o Instagram sobre procedimentos de harmonização facial. A skill deve sempre: - Escrever para mulheres de 28–45 anos interessadas em rejuvenescimento - Usar tom próximo, educativo, sem ser técnico demais - Incluir 1 pergunta para engajar nos comentários - Terminar com CTA para agendar avaliação - Adicionar 5 hashtags relevantes no final Quando eu invocar a skill, pergunte: sobre qual procedimento devo escrever?
Após criar, invoque com: /legenda-harmonizacao — ela pergunta o procedimento e entrega a legenda
Skill 2 · Carrossel educativo
Crie uma skill com isso
Crie uma skill chamada "carrossel-harmonizacao" que escreva o roteiro completo de um carrossel de 6 slides para o Instagram explicando um procedimento de harmonização facial para leigos. Estrutura de cada execução: - Slide 1: título chamativo que gera curiosidade - Slide 2: o que é o procedimento (linguagem simples) - Slide 3: quem se beneficia / indicações - Slide 4: como funciona na prática (sem termos técnicos) - Slide 5: resultado esperado e cuidados pós-procedimento - Slide 6: CTA + chamada para avaliação gratuita Tom: educativo, acolhedor, sem jargão médico. Público: [descreva quem você atende — preencha antes de criar a skill] Quando invocar a skill, pergunte: qual procedimento devo explicar?
Após criar, invoque com: /carrossel-harmonizacao — ela entrega o roteiro dos 6 slides pronto
Skill 3 · Plano de conteúdo mensal
Crie uma skill com isso
Crie uma skill chamada "plano-conteudo-mensal" que monte um calendário de conteúdo de 30 dias para o meu consultório de harmonização facial. Parâmetros fixos da skill: - Procedimentos que ofereço: [liste seus procedimentos — preencha antes de criar] - Frequência: [ex: 4x por semana — preencha antes de criar] - Formatos: feed, reels, stories Para cada post, a skill deve sugerir: - Dia da semana - Formato (feed / reels / stories) - Tema / assunto - Objetivo (educar / engajar / converter) - Gancho de abertura (primeira frase) Resultado entregue em tabela clara. Quando invocar a skill, pergunte: para qual mês devo montar o calendário?
Preencha os procedimentos e frequência antes de criar. Depois: /plano-conteudo-mensal
Skill 4 · Stories com CTA
Crie uma skill com isso
Crie uma skill chamada "stories-cta" que escreva uma sequência de 4 stories para promover um procedimento de harmonização facial. Estrutura fixa de cada execução: - Story 1: pergunta ou enquete para engajar - Story 2: apresentação do problema ou desejo do público - Story 3: apresentação do procedimento como solução - Story 4: CTA direto com link para agendamento Regras: - Tom conversacional, como se fosse uma conversa real - Máximo 120 caracteres por story - Minha persona: [como você se apresenta — preencha antes de criar] Quando invocar a skill, pergunte: qual procedimento devo promover?
Preencha sua persona antes de criar. Depois: /stories-cta — ela pergunta o procedimento
Dica de ouro: após a skill entregar o conteúdo, peça: "Releia como se fosse uma paciente de 35 anos que ainda não me conhece. O que geraria dúvida ou insegurança?" Ela aponta o que você não enxerga por já conhecer o assunto.
Indo além · Integrações avançadas
Geração de imagens
Para criar os visuais do carrossel

Skills como a world-class-carousel conseguem gerar as imagens dos slides automaticamente — mas para isso é necessário conectar com o Google Gemini como provedor de geração de imagem. É uma configuração feita uma vez e libera criação visual dentro do próprio app.

Postagem automática
Para publicar direto nas redes

É possível fazer a Claude postar o conteúdo automaticamente no Instagram, mas exige uma integração com a API da Meta — o que requer criar um aplicativo no painel de desenvolvedor da Meta e conectá-lo ao app. É um processo mais técnico, mas viável. Para a maioria dos consultórios, o fluxo ideal é: Claude cria o conteúdo → você revisa → posta manualmente ou agenda pelo próprio Instagram.

Organização financeira
sem planilheiro

A Claude não é só criação de conteúdo. Ela organiza o lado financeiro do consultório — e cria as ferramentas que você precisar.

O que pedir para a Claude
01
Planilha de controle de atendimentos
Pede para criar uma planilha com: data, paciente (iniciais), procedimento, valor cobrado, forma de pagamento, custo de insumo e lucro líquido. Ela gera o arquivo pronto para usar.
02
Análise de procedimentos mais rentáveis
Joga os dados da planilha na conversa e pergunta: "quais procedimentos me dão mais margem?" Ela cruza os números e responde em texto simples, sem você precisar calcular nada.
03
Modelo de orçamento para pacientes
Pede um template profissional de orçamento para enviar por WhatsApp ou e-mail — com seu nome, procedimento, valor, validade e condições de pagamento.
04
Controle de estoque de insumos
Cria uma planilha de entrada e saída de produtos (fios, ácido, toxina) com alertas de reposição e cálculo de custo por procedimento.
Prompt · Planilha de controle financeiro
Copie, cole e adapte
Crie para mim uma planilha de controle financeiro para um consultório de harmonização facial. Colunas necessárias: - Data do atendimento - Nome do paciente (só iniciais, por privacidade) - Procedimento realizado - Valor cobrado (R$) - Forma de pagamento (Pix / cartão / dinheiro) - Custo dos insumos (R$) - Lucro líquido (calculado automaticamente) Adicione uma aba de resumo mensal com: - Total de atendimentos - Faturamento bruto - Custo total de insumos - Lucro líquido total - Procedimento mais realizado no mês Formato: Google Sheets (me dê as fórmulas prontas)
Pronto para usar — cole direto e envie
Próximo nível: com o Cowork ativado, a Claude consegue ler um arquivo da sua pasta e analisar os dados automaticamente — sem você precisar copiar e colar nada.

Mão na massa:
suas dores na ponta do lápis

Dores comuns na área
💬
Mesmas perguntas no DM
"Dói?" "Quanto custa?" "Quanto dura?" — respondidas manualmente todo dia.
📋
Relatório de evolução
Montar o histórico do paciente e registrar evolução de sessão em sessão.
📩
Follow-up de pacientes
Pacientes que sumiram depois da primeira sessão. Sem tempo para reativar um por um.
💰
Orçamentos repetitivos
Montar o mesmo orçamento dezenas de vezes por semana, sem padronização.
📸
Legenda para antes/depois
Postar resultado sem saber como escrever sem parecer propaganda enganosa.
📅
Confirmação de consultas
Mandar mensagem de confirmação e lembrete para cada paciente manualmente.
Lembra: skill é só um texto com instruções. Se você consegue explicar o processo para um estagiário, você consegue criar uma skill. A Claude só estrutura isso de forma que pode ser repetido.
